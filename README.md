# 수학 세계여행 🌍

초등 고학년~중등 수학을 게임으로 배우는 단일 HTML 웹앱.
9개 과목을 "나라를 여행하며 동물을 모으는" 보드게임 형식으로 학습한다.

## 과목 (9개)

| 과목 | 핵심 내용 | 최고 난이도 |
|------|-----------|-------------|
| 🟰 방정식 | 양팔저울·분배법칙·연립 | 올림피아드 + **이차방정식**(인수분해·근의공식·포물선) |
| 📈 일차함수 | 좌표·함수·기울기 | 두 점으로 식·교점 + **직선과 포물선 결합** |
| ⚖️ 부등식 | 범위·수직선 | 부등호 뒤집힘 + **이차부등식**(포물선 범위) |
| ⚡ 거듭제곱 | 2⁴·지수법칙 | 곱셈/나눗셈/거듭제곱 법칙 |
| 🍕 분수 | 피자 조각·통분·약분 | 대분수 |
| 🟩 소수 | 자릿값·계산 | 유효숫자 반올림 |
| 🔢 큰 수 | 자릿값·반올림 | 지수 표기(a×10ⁿ) |
| 📐 도형 | 각도·삼각형 | 다각형 내각/외각 |
| 📊 그래프 | 막대·꺾은선 | 서술형 분석·미래 예측 |

## 주요 기능

- **이름 입력**: 처음 열면 이름을 받고, 화면·문제에 반영
- **진도 자동 저장**: 과목별로 분리 저장 (브라우저 localStorage)
- **오답 반복 학습**: 틀린 문제 유형을 "연속 3번 정답"까지 반복
  - 나라를 끝낼 때 그 나라의 오답을 복습 코너에서 다시 풀이
- **설정**(우상단 ⚙️): 이름 바꾸기 + 과목별 진도 초기화

## 배포 (Vercel)

빌드 과정이 전혀 없는 순수 정적 HTML이다. `index.html` 하나만 올리면 끝.

```bash
git init
git add index.html
git commit -m "v1.1.0: 이차방정식·이차부등식·그래프결합 + 포물선 시각화"
git remote add origin https://github.com/<아이디>/<저장소>.git
git branch -M main
git push -u origin main
git tag v1.1.0 && git push origin v1.1.0
```

이후 vercel.com → Add New → Project → 저장소 선택 → Deploy.
이후엔 `git push`만 하면 자동 재배포된다.

## 기술 메모 (유지보수용)

- **단일 파일 구조**: `index.html` 안에 9개 게임이 각각 IIFE 모듈로 격리됨
  (`window.GAMES[id] = {start, progress, _game}`). 같은 변수명(GEN/GROUPS/game)을
  써도 충돌하지 않는다. 메뉴가 고른 게임만 `window.game`에 연결된다.
- **진도 저장키**: 과목마다 다름 (`eqgame_v3_`, `funcgame_v1_` 등) + `eunhye` 접미사.
  사용자 이름과 무관하게 기기당 1명 기준으로 저장된다.
- **오답 복습 엔진**: 각 게임의 `state.reviewQueue = {생성기명: 연속정답수}`.
  틀리면 0으로 등록, 맞히면 +1, 3이면 졸업(제거). 나라 마지막 단계에서
  큐가 비어있지 않으면 복습 문제를 추가한다.
- **알려진 제약**: 진도는 기기·브라우저별 저장(서버 없음)이라 기기 간 동기화는 안 됨.
  한 기기를 여러 명이 쓰면 진도가 공유된다(설정에서 과목별 초기화로 대응).

## 향후 확장 아이디어

- 공통 엔진을 `engine.js`로 분리하고 각 과목을 `{GEN, GROUPS, 그리기함수}` 설정으로
  주입하면 유지보수가 쉬워짐 (현재는 9개 게임 코드가 한 파일에 인라인).
- 서버 저장(로그인)으로 기기 간 진도 동기화.
- 전체 오답을 한곳에 모아 복습하는 "오답 노트" 화면.
