은혜의 그래프 탐험

점 하나에서 시작해서, 멋진 곡선까지! 천천히 같이 가보자 😊

STEP 1

📍모눈종이와 좌표

'좌표'는 점이 어디 있는지 알려주는 주소야.

🤔 먼저, 왜 모눈종이를 그릴까?
모르는 수(미지수)가 1개면 모눈종이가 필요 없어!
예) x + 1 = 3 → x는 2.
이런 건 수직선 하나에서 앞뒤로 찾으면 끝이야.

모눈종이가 필요한 진짜 이유는 모르는 수가 2개일 때야!
그 둘을 x, y라고 불러.

모눈종이엔 가로줄과 세로줄이 있어.
• 가로 길 = x
• 세로 길 = y
점은 (가로 칸, 세로 칸)으로 말해.
예) (3, 2) → 오른쪽 3칸, 위로 2칸! 🎯

😲 놀라운 발견!
x랑 y 두 개를 어떻게 그릴지 고민하다가,
모눈종이에 점을 콕콕 찍어봤더니…
점들이 직선으로 쭉 이어졌어! ✨ (STEP 2에서 더 놀아보자)

• x는 이렇게 커지고 작아지고
• y도 이렇게 커지고 작아지고
모눈종이에 그리면 모든 (x, y)를 한눈에 표시할 수 있어. 대단하지? 👍

🎯 가로축과 세로축이 만나는 점 = 원점 (0, 0).
x도 0, y도 0인 자리 → 여기서 출발하는 거야! 🚀
👉 아래 모눈종이를 톡 눌러 점을 찍어봐. 주소(좌표)를 알려줄게!
모눈종이를 눌러서 점을 찍어보자!
조금 더! 좌표 = 자리를 표시한 것. 영어로는 coordinates(코오디네이츠).
STEP 2

✏️규칙이 그래프가 돼

그래프는 사실, 규칙을 지키는 점들이 모인 것이야.

규칙 하나를 보자: y = 2x + 1
뜻은 "x를 2배 하고, 1 더하기"!
x에 숫자를 하나씩 넣어볼까?

x=0y = 2×0 + 1 = 1 → 점 (0, 1)
x=1y = 2×1 + 1 = 3 → 점 (1, 3)
x=2y = 2×2 + 1 = 5 → 점 (2, 5)

이 점들을 쭉 이으면? 선이 돼!
아래 버튼을 누르면 점이 하나씩 찍혀.

버튼을 눌러 x = 0 부터 넣어보자!
조금 더! x를 넣으면 y가 딱 하나 나오는 규칙 → 중학교에서 함수라고 배워. "숫자를 넣으면 답이 나오는 상자" 📦 (영어로 function).

🔑직선 그래프의 비밀번호 2개

그래프를 식으로 쓰면 y = 2x + 2 처럼 "y = ○○○" 모양이 돼.
x를 넣으면 y가 나오는 거지.

놀라운 사실! 직선은 딱 2가지만 알면 그릴 수 있어.
• 기울기
• y절편
하나씩 만져보자 👇

① 기울기 = 선이 얼마나 가파른지.
y = 2x 의 기울기를 바꿔봐:

버튼을 눌러 기울기를 바꿔봐!
📐 기울기 규칙!
• + → 오른쪽 위로 ↗
• − → 오른쪽 아래로 ↘
• 숫자 클수록 가팔라져 🚀
• ½처럼 작으면 천천히 🐌
• 0이면 납작한 가로선

② y절편 = 선이 세로축과 만나는 높이.
(x=0일 때의 y값이야.)
y = 1x + 0 의 절편을 바꿔봐:

버튼을 눌러 y절편을 바꿔봐!
🎯 선이 위아래로 통째로 움직이지?
• 절편 3 → (0, 3)에서 만나
• 절편 −2 → (0, −2)에서 만나
가로축과 만나는 곳은 x절편이라고 해!
조금 더! 기울기 = 영어로 slope(슬로프).
절편 = 축을 끊고 지나가는 자리. y축을 끊으면 y절편, x축을 끊으면 x절편!

🎮 이제 게임!
기울기랑 y절편을 알면 식만 보고 그래프를 찾을 수 있어.
아래 식에 맞는 그래프를 골라봐!

위 식에 맞는 그래프 번호를 눌러봐!
🏆 대단해 은혜!
이제 식만 봐도 그래프가 머릿속에 그려지지?
어려운 문제도 답이 어디 있는지 눈에 보여. 진짜 강력한 무기가 생긴 거야! 😎✨

🎛️나만의 직선 만들기 놀이판

이제 기울기랑 y절편을 동시에 움직여봐!
손잡이를 좌우로 끌면 세상 모든 직선을 만들 수 있어.

y = 1x + 0
기울기 (a) 1
y절편 (b) 0
손잡이를 움직여봐! 빨간 점은 y절편, 파란 점은 x=1일 때의 위치야.
💡 두 손잡이로 무엇이든 만들 수 있어!
• 기울기(a) → 방향·가파름
• y절편(b) → 높이
이게 직선의 전부야!

🧩답이 여러 개일 수도 있어?

지금까지 푼 문제는 답이 하나였지? 그런데 이런 문제는 좀 달라.

은혜가 젤리(x개)랑 사탕(y개)을 샀어.
둘을 합쳐서 5개! 식으로 쓰면:

x + y = 5

x랑 y는 아직 모르는 수 두 개야.
젤리랑 사탕이 각각 몇 개일까? 아래 버튼으로 맞춰보자!

젤리 개수를 골라봐!
🤔 봤지? (0,5) (1,4) (2,3) (3,2)...
답이 여러 개야!
조건이 하나뿐이라 딱 하나로 못 정해. 모르는 수가 2개라서 그래.
조금 더! 모르는 수가 두 개(x, y)인 식 → 중학교에서 미지수가 2개인 방정식이라고 배워.
미지수 = 아직 모르는 수 (영어로 unknown).
STEP 3

🕵️조건이 하나 더! 딱 정해지는 답 찾기

답이 여러 개라 답답했지? 그럼 힌트(조건)를 하나 더 받아보자!

📖 이야기:
은혜가 젤리(x)랑 사탕(y)을 합쳐서 5개 샀어.
엄마: "젤리가 사탕보다 몇 개 더 많아?"
은혜: "1개 더 많아!"

🤔 먼저 생각해보기:
이야기에 중요한 말이 두 개 숨어 있어.
① "합쳐서 5개"
② "젤리가 1개 더 많아"
그래서 식도 두 개! 같이 만들어보자.

✍️ 첫 번째 문장을 식으로:
"합쳐서 5개" — '합친다'는 더하기겠지?
빈칸에 알맞은 걸 골라봐!

x 　?　 y = 5

✍️ 두 번째 문장을 식으로:
"1개 더 많아" — 젤리에서 사탕을 빼면 차이가 1!
빈칸을 채워봐!

x 　?　 y = ?
먼저 가운데 기호:
그 다음 오른쪽 숫자:

🎉 두 문장이 식 두 개가 됐어! 정리하면:

조건1합쳐서 5개 → x + y = 5
조건2젤리가 1개 더 많아 → x − y = 1

이제 두 조건을 동시에 만족하는 (젤리, 사탕)을 찾아야 해. 어떻게 찾을까?

🐢 방법 ① 무작정 넣어보기
숫자를 넣어서 둘 다 맞는지 확인! 직접 해볼까?

젤리 개수를 골라서 두 조건이 다 맞는지 확인해봐!
😮‍💨 하나씩 다 넣어보면 너무 귀찮지?
숫자가 컸으면 더 오래 걸려.
그래서 더 똑똑한 방법이 필요해!

🎨 방법 ② 그림으로 보기
STEP 2처럼 점을 찍어 이으면 선이 돼.

그런데 x + y = 5는 점 찍기가 불편해.
그래프는 "y = ○○○" 모양이 제일 편하거든.
그래서 y만 남겨보자!

⚖️ 비밀 규칙: 식은 양팔저울!
왼쪽에 한 건 오른쪽에도 똑같이 해야 균형이 안 깨져.

먼저 x + y = 5 에서 y만 남기기 — 한 줄씩 눌러봐:

지금x + y = 5 … 왼쪽에 방해되는 x가 있네. 이걸 없애자!
1왼쪽의 x를 없애려면? 양쪽에서 똑같이 x를 빼! → x + y − x = 5 − x
2왼쪽은 x − x = 0 이라 사라지고, y만 남아 → y = 5 − x 🎉

같은 방법으로 x − y = 1 도 y만 남겨보자:

지금x − y = 1 … 이번엔 y 앞에 −가 붙어서 좀 까다로워.
1양쪽에서 똑같이 x를 빼면 → − y = 1 − x
2아직 y 앞에 −가 있네. 양쪽에 똑같이 −1을 곱하면 부호가 뒤집혀 → y = x − 1 🎉

이제 STEP 2처럼 x에 숫자를 넣어 점을 찍어보자! 먼저 파란 선 (y = 5 − x):

다음은 분홍 선 (y = x − 1):

위 숫자 버튼으로 점을 찍어보자!
💡 • 파란 선 = "합쳐서 5개"인 답들
• 분홍 선 = "1개 더 많은" 답들
두 조건을 다 만족하는 답은?
두 선이 만나는 딱 한 점이야! 🎯
조금 더! 양쪽에 똑같이 더하거나 빼서 글자를 옮기는 걸 → 중학교에서 이항이라고 배워.
"넘기면 부호가 바뀐다"고 외우지만, 사실은 양팔저울처럼 양쪽에 똑같이 한 것이야! (영어로 transposition)

✏️ 방법 ③ 계산으로 깔끔하게 — 그림으로 감을 잡았으니, 계산으로도 확인해보자.

⚖️ "두 식을 더해도 돼?"
x − y = 1 은 "x−y 랑 1 이 같다"는 뜻이야.
그러니까…
• 왼쪽에 (x−y) 더하기
• 오른쪽에 1 더하기
→ 사실 양쪽에 똑같은 값을 더한 거지! ⚖️
그래서 균형이 안 깨지고 답도 맞아.

자, 그럼 한 줄씩 눌러봐:

1두 식을 위아래로 더하기 — 왼쪽끼리, 오른쪽끼리: (x+y) + (x−y) = 5 + 1
2왼쪽을 정리하면 +y 와 −y 가 만나 0이 돼서 y가 쏙 사라져! → 2x = 6
3양쪽을 똑같이 2로 나누면 → x = 3 (젤리 3개!)
4x=3을 x+y=5에 넣으면 → y = 2 (사탕 2개!) → 답 (3, 2) ✅
🌟 핵심! 그림에서 두 선이 만난 점 (3, 2)랑 계산 답 (3, 2)가 똑같지?
그래서 "두 조건의 답 = 두 선이 만나는 점"이야!
조금 더! 답을 찾으려고 푸는 식 = 방정식 (영어로 equation).
두 식을 한 묶음으로 같이 푸는 건 연립방정식이라고 배워.
두 그래프가 만나는 점은 교점 (영어로 intersection point).
STEP 4

🥣휘어지는 곡선 (2차)

x를 두 번 곱하면(x × x), 선이 그릇처럼 휘어져!

규칙은 y = x × x (= x²).
음수도 두 번 곱하면 양수가 되는 게 비밀!
x에 −3부터 3까지 점을 하나씩 찍어봐.
점이 많아질수록 진짜 모양이 드러나!

위 숫자 버튼을 눌러서 점을 하나씩 찍어봐!
🔎 U자 그릇 모양이 보이지?
맨 아래 빨간 점 (0, 0)이 가장 낮은 곳 = '꼭짓점'.
−3과 3은 둘 다 y가 9라서 좌우가 똑같아!
조금 더! x를 두 번 곱한 식 = 이차식.
이 그릇 모양 곡선은 포물선이라고 배워 — 공을 던졌을 때 날아가는 길 모양이야 ⚾ (영어로 parabola).
STEP 5

🐍구불구불 곡선 (3차)

x를 세 번 곱하면(x × x × x), 곡선이 구불 한 번 더 휘어져!

규칙은 y = x × x × x (= x³).
−3부터 3까지 점을 하나씩 찍어봐.
모일수록 구불구불 S자가 드러나!
아까 그릇이랑 어떻게 다른지 봐.

숫자 버튼을 눌러서 S자 모양을 만들어봐!
🔎 가운데 빨간 점 (0, 0)을 잘 봐!
여기서 휘는 방향이 바뀌어 (아래로 → 위로).
그릇(2차)엔 없던 점이 구불 곡선(3차)엔 생겨. 이게 핵심이야!
조금 더! 곡선이 휘는 방향이 바뀌는 점 = 변곡점이라고 배워. (영어로 inflection point)

🎢자유 놀이터 (3D)

이번엔 입체로! x, y, z 세 방향이 있는 3차원 세상이야.

지금까진 평평한 모눈종이(2차원)였어.
여기에 높이(z)를 더하면 산·골짜기 같은 입체가 돼!
그림을 마우스로 빙글빙글 돌려보고, 버튼으로 모양도 바꿔봐!

z = x × x + y × y
